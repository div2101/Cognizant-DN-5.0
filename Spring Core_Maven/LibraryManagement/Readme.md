# Exercise 4 - Maven Project Setup for Library Management

## Objective

This exercise demonstrates how to create and configure a Maven project with Spring dependencies.

---

## Features Configured

### 1. Spring Dependencies
- Spring Context (Core container)
- Spring AOP (Aspect-Oriented Programming)
- Spring Web MVC (Web framework)

---

### 2. Maven Compiler Plugin
- Java version set to **1.8**
- Ensures compatibility across environments

---

## How to Build Project

```bash
mvn clean compile
```

---

## Project Setup Steps

1. Create Maven project (LibraryManagement)
2. Add Spring dependencies in `pom.xml`
3. Configure Maven Compiler Plugin
4. Build project using Maven commands

---

## Expected Output

```
BUILD SUCCESS
```

---

## Learning Outcome

- Understand Maven project structure
- Add and manage Spring dependencies
- Configure Java version using Maven plugin
